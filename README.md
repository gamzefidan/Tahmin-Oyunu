Tahmin Oyunu

Bu proje, React kullanılarak geliştirilmiş basit bir sayı tahmin oyunudur.
Kullanıcıdan 1 ile 100 arasında bir sayı tahmin etmesi istenir.
Uygulama, tahminin doğru olup olmadığını bildirir.

Özellikler

\*1–100 arası rastgele sayı üretimi

\*Tahmin sonucu anlık geri bildirim

\*Yeni oyun başlatma seçeneği

\*Modern, responsive tasarım

\*Arka planda Matrix tarzı sayılar animasyonu

Kullanılan Teknolojiler

\*React

\*CSS

Projeyi klonla:

```bash
git clone https://github.com/kullanici-adin/tahmin-oyunu.git
```

Proje dizinine gir:

```bash
cd tahmin-oyunu
```

Bağımlılıkları yükle:

```bash
npm install
```

Uygulamayı Başlat:

```bash
npm start
```
